회의실 사용기준(2021년 1월 11일 적용)

· 회의실 예약은 실 사용시간을 기준으로 예약해야 함
· 회의실 예약 후 이용 1시간 전 예약자에게 예약 확정 안내 문자(SMS)가 발송되며 문자 안내에 따라
   회의실 예약 확정을 진행해야 함 (이용시작시간 +30분까지만 가능)
· 회의실 예약 미 확정 또는 예약 확정 후 회의실 미 이용 적발 시 페널티 점수 부여
· 프로젝트 실행을 위한 회의실 장기 사용 / 물품 보관 용도로 예약을 금지함

  • 페널티 운영기준
  • TV모니터 연결방법
  • 스튜디오실 이용방법

· 예약 후 미 확정 시 1점, 예약 확정 후 미 이용 시 2점
· 누적 페널티 5점 도달 시 한 달간 회의실 예약 불가
    - 누적 5점 이후부터는 2점 추가마다 한 달간 예약 불가 페널티 적용
     - 누적 페널티 점수는 연 초가 되면 0점으로 초기화
· 회의실 예약관련 문의 : 경영기획실(669)

1. TV모니터 리모컨 우측 상단에 TV/외부입력 선택 후 Remote Access 선택
2. 연결 노트북에서 우측 하단 알림 선택 후 연결 선택
3. 해당 회의실의 TV모니터 선택 후 연결 완료

· 이용 전 경영기획센터 방문 후 스튜디오실 키를 받아 사용 가능
· 사용 완료 후 반드시 반납
